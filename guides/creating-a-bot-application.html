Creating a Bot Application


Head on over to the Discord Developer Application page

Select New App

Give your app a name (a App Icon & Description is not nessasary)

When you have completed giving the needed information select Create App

This should now present you with your new Bot App. Scroll down to the bot section.

Select Create a Bot User. This should present you with a confirmation box asking for confimation, select Yes, do it!

You should now see your bot section has now updated and look, you can see your bot token!



Disclaimer: Do not give your Bot Token to ANYONE!

Giving someone else access to your bot token can allow other users to abuse the Discord API on your behalf which in result may get your account banned.
