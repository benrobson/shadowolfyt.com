Introduction

There is no real 'commands' for this bot. This bot is basically a meme bot. It contains gifs or gifs (I don't know how it's pronouned) that users can call using a command (or in this case a tag).

But, there is some documentation about some of the tags and what they display and use. You can click the hyperlinked text in the tags sidebar.


tags

The tags command is the main command to list all tags the user can use.


jsfirst

The jsfirst command is command that users can see information about how to get started with JavaScript.


jsfaq

The jsfirst command is command that users can see the common questions that people ask developing, asking questions and debugging code with JavaScript.


joinroles

The jsfirst command is command that displays information about why auto join roles (that some bots and guild leaders use) are dangerous and bad.


2fa

The 2fa command is displays information about what 2fa (or Two Factor Authentication) is and why it is important.
